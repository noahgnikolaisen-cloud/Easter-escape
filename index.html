🔧 Firebase Setup Required

To enable multiplayer, you need to connect this game to a free Firebase project. It takes about 3 minutes:

  1. Go to console.firebase.google.com
  2. Create a new project (free)
  3. Go to Build → Realtime Database → Create database
  4. Go to Project Settings → Your apps → Add web app
  5. Copy the firebaseConfig object
  6. Open this HTML file and paste it in the firebaseConfig section at the top

Also set Realtime Database Rules to allow read/write while playing:
{ "rules": { ".read": true, ".write": true } }
